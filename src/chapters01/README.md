# 第一章 ECMAScript 6 简介

`ES6`的目标，是使得`JavaScript`语言可以用来编写大型的复杂应用程序，成为企业级开发语言。

`ECMAScript`是`JavaScript`语言的国际标准，`JavaScript`是`ECMAScript`的实现。
